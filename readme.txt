Анализ сервиса аренды самокатов GoFast.

Исходынми данными являются 3 датафрейма, содержащие информацию о пользователях,типе их подписки,дистанции,продолжительности поездки,дате начала сессии и тарифном плане.

План проекта:
    1. Выполнить загрузку и предобработку данных

    2. Провести исследовательский анализ данных с построением графиков.

    3. Произвести расчет выручки от пользователей с подпиской и без.

    4. Проверить гипотезу: тратят ли пользователи с подпиской больше времени на поездки.

    5. Проверить гипотезу: расстояние, которое проезжают пользователи с подпиской за одну поездку, не превышает 3130 метров.

    6. Проверить гипотезу о том, будет ли помесячная выручка от пользователей с подпиской по месяцам выше, чем выручка от пользователей без подписки. 

    7. Итоговый вывод проделанном анализе с рекомендациями по улучшению работы сервиса,способствующие его расширению и повышению выручки.
